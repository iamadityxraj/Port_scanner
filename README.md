# Python Port Scanner

A simple Python-based TCP Port Scanner that identifies open ports on a target host using Socket Programming.

## Features

* Scan TCP ports on a target host
* Detect open ports
* Uses Python Socket Programming
* Beginner-friendly networking project

## Technologies Used

* Python
* Socket Programming
* TCP/IP Networking

## How to Run

1. Clone the repository
2. Run the script:

python port_scanner.py

3. Enter the target IP address when prompted

## Learning Outcomes

* Understanding TCP/IP communication
* Working with sockets in Python
* Network scanning fundamentals
* Basic cybersecurity and networking concepts
